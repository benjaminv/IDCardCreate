# IDCardCreate
生成身份证，身份证号码校验

1、生成身份证

依次选择省-市-县-生日后点击即可生成身份证号码。

关于15~17位是判断统一地区时间不同的小孩和性别，不是强校验的，随机生成即可。

理论上顺序码的奇数分配给男性，偶数分配给女性。
